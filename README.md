# SysPulse (In Progress)

SysPulse is a system health and performance monitoring platform designed to
collect, store, and analyze metrics across distributed machines.

This project is currently under active development.

## Current Capabilities
- Local system metrics collection
- Persistent storage of metrics
- Modular architecture for future expansion

## Planned Features
- Distributed agent support
- Threshold-based alerting
- Historical analytics and anomaly detection
- CLI and dashboard interfaces

## Tech Stack
- Python
- SQLite
- psutil
